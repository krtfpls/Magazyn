# WMService on Angular

My own, small application for simple warehouse management for field service staff. One of my first project, so please bear with it.

## Technologies:
* API: C# with .Net 7.0
  * EntityFramework
  * CQRS Pattern
  * MediatR
  * FluentValidator
  * JWT Token
  * AutoMapper
  * Postgres DB
  
* FrontEnd: Angular ver. 14
  * ngx-bootstrap
